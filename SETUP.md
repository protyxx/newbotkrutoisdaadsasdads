# ИНСТРУКЦИЯ ПО НАСТРОЙКЕ БОТА

## Шаг 1: Получите токен бота

1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/newbot`
3. Следуйте инструкциям (придумайте имя и username для бота)
4. Скопируйте полученный токен

## Шаг 2: Узнайте свой Telegram ID

1. Найдите бота @userinfobot в Telegram
2. Нажмите /start
3. Скопируйте ваш ID (число)

## Шаг 3: Настройте основной канал

1. Создайте канал (или используйте существующий)
2. Добавьте вашего бота в канал как администратора
3. Дайте боту права:
   - Добавление участников
   - Просмотр списка участников
4. Узнайте ID канала:
   - Перешлите любое сообщение из канала боту @userinfobot
   - Скопируйте ID канала (начинается с -100)

## Шаг 4: Настройте обязательные каналы

Для каждого канала, на который должны подписаться пользователи:

1. Добавьте бота в канал как администратора
2. Дайте права на просмотр участников
3. Узнайте ID канала (как в шаге 3)

## Шаг 5: Заполните config.py

Откройте файл `config.py` и заполните:

```python
# Токен от @BotFather
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"

# Ваш Telegram ID
ADMIN_ID = 123456789

# ID основного канала (куда подают заявки)
TARGET_CHANNEL_ID = -1001234567890

# Название канала для пользователей
TARGET_CHANNEL_NAME = "Название вашего канала"

# Обязательные каналы (уже настроены ваши каналы)
REQUIRED_CHANNELS = [
    {
        "id": -1003579431986,
        "name": "ParanoicDLC",
        "url": "https://t.me/ParanoicDLC"
    },
    {
        "id": -1003878098115,
        "name": "Второй канал",
        "url": "https://t.me/+hKspschwnmQzOTgy"
    }
]
```

## Шаг 6: Запустите бота

```bash
cd "C:\Users\steam\OneDrive\Рабочий стол\botik"
python bot.py
```

## Как это работает:

1. Пользователь подает заявку на вступление в ваш канал
2. Бот отправляет ему сообщение со ссылками на обязательные каналы
3. Пользователь подписывается и нажимает "Проверить подписки"
4. Бот проверяет подписки и автоматически одобряет заявку

## Команды для администратора:

- `/stats` - статистика бота
- `/broadcast` - рассылка сообщения всем пользователям
- `/cancel` - отмена операции

## ВАЖНО:

- Бот должен быть администратором во ВСЕХ каналах (основном и обязательных)
- У бота должны быть права на добавление участников
- Не забудьте установить зависимости: `pip install -r requirements.txt`
